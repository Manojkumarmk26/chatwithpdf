================================================================================
                              API ENDPOINTS
================================================================================

1. POST /api/summary/generate
   - Generate and save summary for a document
   - Input: filename, session_id, user_id
   - Output: summary text + metadata

2. POST /api/summary/combine
   - Combine multiple summaries into one
   - Input: filenames[], session_id, user_id
   - Output: combined summary + metadata

3. POST /api/summary/condense
   - Create executive summary from long summary
   - Input: summary_text
   - Output: condensed summary (3 paragraphs)

4. GET /api/summary/retrieve/{filename}
   - Retrieve previously saved summary
   - Input: filename
   - Output: summary text + metadata

5. GET /api/summary/list
   - List all saved summaries
   - Input: none
   - Output: array of summaries with metadata
